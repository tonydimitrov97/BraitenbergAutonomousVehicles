Detailed Description

The main class for the simulation of a 2D world with many entities running around.

While GraphicsArenaViewer handles the graphics, Arena handles all the data and all the entities (it is the model of MVC). It manages the interaction among the entities in the Arena.

Parameters
entThe entity to check.
[out]Anentity type signifying wall (e.g. kRightWall). kUndefined if no collision.

The checked entity's position will be updated to a "back-off position" so that it won't get stuck into a wall. The calculation of the "back-off position" is technically not accurate, but good enough for our purpose.
