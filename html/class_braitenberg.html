Detailed Description

This parent class for all of the possible robot behaviors.

The three possible subclasses as of right now are Fear, Aggression, Explore These classes override the CalcWheelVelocity and CalcWheelVelocityWFood methods. Based upon the circumstances this controls the robot's behavior towards food and light.
